# Career Roadmap Generator

A modern and interactive web application that helps users explore structured learning roadmaps for different career paths in technology.

## 🚀 Features

* Career selection from multiple domains:

  * Web Development
  * AI & Machine Learning
  * Data Science
  * Cyber Security
  * Cloud Computing
  * UI/UX Design
* Beginner, Intermediate, and Advanced learning stages
* Recommended skills and tools for each stage
* Suggested projects to build practical experience
* Progress tracking with a visual progress bar
* Responsive design for desktop and mobile devices
* Dark/Light mode support
* Modern user interface with animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 How to Use

1. Open the website in your browser.
2. Select your desired career path.
3. Explore the roadmap stages.
4. Follow the recommended skills, tools, and projects.
5. Track your learning progress.
